Usage Guide & API Documentation

Channel Card

Simply add a YouTube channel username to generate a card:

https://youtubecard.vercel.app/?username=MrBeast

Video Card

Generate a card for any YouTube video using its ID or URL:

https://youtubecard.vercel.app/?video=dQw4w9WgXcQ

With Theme

Choose between light and dark themes:

https://youtubecard.vercel.app/?username=MrBeast&theme=dark

GitHub README Integration

Perfect for showcasing your YouTube presence in README files:

<img src="https://youtubecard.vercel.app/?username=MrBeast" alt="@MrBeast" style="width: 300px;" />

Or video cards:

![Video](https://youtubecard.vercel.app/?video=dQw4w9WgXcQ)

Advanced Customization

Customize colors, fonts, and shadows to match your brand:

https://youtubecard.vercel.app/?username=MrBeast&bgColor=rgba(15,15,15,1)&textColor=%23FFFFFF&subtleTextColor=%23AAAAAA&extraColor=%23FF0000&shadowColor=rgba(255,0,0,0.2)&fontFamily=system-ui

Available Parameters

Customize your cards with these URL parameters:

  • username - YouTube channel username (for channel cards)
  • video - YouTube video ID (for video cards)
  • theme - light or dark
  • bgColor - Background color (rgba or hex)
  • textColor - Main text color
  • subtleTextColor - Secondary text color
  • extraColor - Accent color for subscriber/view count
  • shadowColor - Box shadow color
  • fontFamily - Font family name
